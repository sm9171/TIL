# Clean Code-Rovert C.Martin

완독: 진행중

로버트 C.마틴의 클린코드를 읽고 인상깊은 구문 정리

  • 46p. 코드는 위에서 아래로 이야기처럼 읽혀야 조음. 최대한 추상화 수준이 하나인 함수를 구현해라. 위에서 아래로 프로그램을 읽으면 추상화 수준이 한 단계씩 낮아짐 (추상화 수준이 하나 - 라는 표현에 살짝 충격. 내가 모호하게 생각하고 있던 판단 기준을 명료하게 정리함)

  • 52p. 플래그 인수 피하기. 나는 함수 안에 여러가지를 처리할거예요~ 라 말하는거임.

    // bad
render(isFood);

// good
if (isFood) {
  renderFood();
} else {
  renderOther();
}
